# Portfolio Website& Music Lesson Subscription Platform

### Deployed Live-Demo Website Link: https://mpark-se.quest/

## Description:

A full-stack web application enabling music lesson subscriptions with automated scheduling and payment processing

Technologies:
- Frontend: React
- Backend: Node.js, Express.js
- Database: PostgreSQL (hosted on Neon)
- Payment Processing: Stripe
- Scheduling: Cal.com
- Deployment: Railway (Frontend/Backend), custom domain with SSL

## Future Additions:

- Google Maps embed after Google Business is approved
- More teaching and subscription options
